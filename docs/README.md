# 欢迎使用韦二伯格的开源电子书模板

这是一个基于 VuePress 构建的开源电子书模板，它具有以下特点：

## 特点

- **简单易用**：只需要会写 Markdown 文档，就能创建美观的电子书
- **功能丰富**：自动生成目录、搜索功能、响应式设计
- **版本控制**：基于 Git 的版本控制，轻松管理内容更新
- **自动部署**：通过 GitHub Actions 自动构建和部署

## 如何使用

请查看 [指南](/guide/) 了解如何使用这个模板创建你自己的电子书。
